Pharmacy Admin Project

This is a Pharmacy Admin Website. Here, we can view the medicines that are ordered by customers through pharmacy client website. Employees will then prepare and deliver the medicines to address provided to customers We can also keep a tab on the imports flow of the shop and let employees to check whether imported products have arrived or not. Finally we have various reports such as sale report, employee report, customer report, rating report, product report, etc... The Website has been developed using React and Scss with MongoDB as database.
